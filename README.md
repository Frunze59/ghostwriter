# Ghostwriter

AI-powered content generation platform. Supports Blog Posts, Emails, Short Stories, and Social Media posts — generated by Claude (Anthropic) and streamed to the browser in real time.

---

## Setup (Docker — recommended)

The only requirement is [Docker Desktop](https://www.docker.com/products/docker-desktop/).

**1. Get a free Anthropic API key**

Go to [console.anthropic.com](https://console.anthropic.com/), create an account, and copy your API key.

**2. Clone and configure**

```bash
git clone https://gitea.kood.tech/sergiufrunze/ghostwriter.git
cd ghostwriter
cp .env.example .env
```

Open `.env` and paste your key:

```
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

**3. Run**

```bash
docker compose up --build
```

- App: http://localhost
- API: http://localhost:3001
- Health check: http://localhost:3001/health

---

## Setup (local dev, no Docker)

**Requirements:** Node.js 20+

```bash
# Backend
cd backend
npm install
npm run dev        # runs on http://localhost:3001

# Frontend (separate terminal)
cd frontend
npm install
npm run dev        # runs on http://localhost:5173
```

The `.env` file must be in the `backend/` directory for local dev, or at the project root for Docker.

## Usage

1. Select a content type (Blog Post, Email, Story, Social Media)
2. Fill in the form — each field has a label and placeholder explaining what's expected
3. Click **Generate** — content streams to the screen word by word
4. Edit the output directly in the editor
5. Export as **Markdown**, **Plain Text**, or **PDF**

---

## API endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/generate` | Full pipeline — streams SSE response |
| `POST` | `/api/validate` | Dry run — returns normalized input, no AI call |
| `GET`  | `/api/content-types` | Field schemas for all content types |
| `GET`  | `/health` | Server status + key loaded check |

## Content types

| Type | Input | Output |
|------|-------|--------|
| Blog Post | Topic, audience, word count, tone, SEO focus, expertise level | Titled article with intro, sections, conclusion |
| Email | Purpose, recipient context, key points, tone, urgency, CTA | Subject line + email body |
| Story | Genre, characters, setting, length, style, mood | Complete short story with narrative structure |
| Social Media | Product/service, platform, goal, tone, CTA | Platform-optimized post with hashtags |
